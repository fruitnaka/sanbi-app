STEP 1 ｜ Firebase Realtime Database を作る

1Googleアカウントでログイン
  • ブラウザで firebase.google.com を開く
  • 右上「コンソールへ移動」をクリック
  • Googleアカウントでログイン（教会のGmailでOK）
2プロジェクトを作成
  • 「プロジェクトを追加」をクリック
  • プロジェクト名を入力（例：sanbi-renraku）
  • Googleアナリティクスは「無効」でOK → 「プロジェクトを作成」
💡 プロジェクト名は半角英数字・ハイフンのみ使えます
3Realtime Database を有効にする
  • 左メニュー「構築」→「Realtime Database」をクリック
  • 「データベースを作成」ボタンを押す
  • ロケーション：asia-southeast1（シンガポール） を選択
  • セキュリティルール：「テストモードで開始」 を選択 → 「有効にする」
⚠️ テストモードは30日間有効です。期限が来たらルールを再設定してください（下記参照）
4データベースURLをコピー
  • Realtime Databaseの画面上部にURLが表示される
https://sanbi-renraku-default-rtdb.asia-southeast1.firebasedatabase.app
  • このURLをメモ帳にコピーしておく（あとでアプリに貼り付けます）
💡 URLは「データ」タブを開くといつでも確認できます

🔒 セキュリティルール（30日後に設定）

Realtime Database → 「ルール」タブ → 以下を貼り付けて「公開」：
{ "rules": { ".read": true, ".write": true } }
⚠️ このルールは誰でも読み書きできます。教会内グループ用途では十分ですが、URLを外部に漏らさないようにしてください

STEP 2 ｜ GitHub Pages でアプリを公開する

HTMLファイルをインターネット上に置くことで、URLを開くだけで使えるようになります。

1GitHubアカウントを作る
  • github.com を開く → 「Sign up」
  • メールアドレス・パスワード・ユーザー名を設定（無料プランでOK）
💡 すでにGitHubアカウントをお持ちの方はこの手順は不要です
2新しいリポジトリを作る
  • ログイン後、右上「＋」→「New repository」
  • Repository name：sanbi-app（何でもOK）
  • 「Public」を選択（Privateだと公開できません）
  • 「Add a README file」にチェック → 「Create repository」
3HTMLファイルをアップロード
  • 作成したリポジトリのページで「Add file」→「Upload files」
  • praise-group.html をドラッグ＆ドロップ
  • ファイル名を index.html に変更する（重要！）
⚠️ ファイル名が index.html でないとURLで直接開けません。アップロード後にファイル名を変更するか、最初から index.html という名前で保存してください
  • 「Commit changes」ボタンを押す
4GitHub Pages を有効にする
  • リポジトリの「Settings」タブ → 左メニュー「Pages」
  • Source：「Deploy from a branch」
  • Branch：main / /(root) → 「Save」
  • 数分後、以下のようなURLが表示される：
https://あなたのユーザー名.github.io/sanbi-app/
💡 公開まで1〜3分かかります。ページを更新して確認してください

STEP 3 ｜ メンバーに共有する

1アプリを開いてFirebaseを設定
  • GitHub PagesのURLをブラウザで開く
  • 初回起動時に設定画面が表示される
  • STEP1でコピーした Firebase URL を貼り付ける
  • 自分の名前を入力 → 「接続して始める」
💡 2回目以降は設定画面は表示されません（端末に保存されます）
2LINEでURLとFirebase URLを共有
LINEグループに以下のメッセージを送るだけです：
【賛美グループ連絡アプリ】 アプリURL： https://あなたのユーザー名.github.io/sanbi-app/ 初回のみ以下のURLが必要です： https://sanbi-renraku-default-rtdb.asia-southeast1.firebasedatabase.app ↑このURLを開いて、設定画面に貼り付けてください。 名前を入力して「接続して始める」を押せば完了です✦
💡 ホーム画面に追加しておくとアプリのように使えます（Safari → 共有 → ホーム画面に追加）

よくある質問・トラブル

「同期中」にならない
  • Firebase URLが正しいか確認（末尾に / は不要）
  • Firebaseのルールが「テストモード」になっているか確認
  • インターネット接続を確認
Androidで画面が崩れる
  • Chrome最新版で開いてください（Safariと同等の表示になります）
設定をやり直したい
  • Chromeの場合：設定 → サイトの設定 → データを削除
  • Safariの場合：設定 → Safari → 詳細 → Webサイトデータ → 該当サイトを削除
  • その後アプリを再読み込みすると設定画面が表示されます
HTMLファイルを更新したい
  • GitHubのリポジトリで index.html をクリック → 鉛筆アイコン（編集）
  • または「Add file」→「Upload files」で上書きアップロード
  • Commit後、数分で全員の画面に反映されます
